# Dialogs, Alerts and Panels

The Flet documentation page titled "Dialogs, Alerts and Panels" outlines various components for displaying interactive messages and panels within a Flet application.

The available components include:

*   **AlertDialog**: A standard Material Design alert dialog.
*   **Banner**: A prominent message requiring user interaction to be dismissed.
*   **BottomSheet**: A modal sheet that appears from the bottom of the screen.
*   **DatePicker**: A Material Design dialog for selecting a date.
*   **SnackBar**: A lightweight message that is briefly displayed at the bottom of the screen.
*   **TimePicker**: A Material Design dialog for selecting a time.
