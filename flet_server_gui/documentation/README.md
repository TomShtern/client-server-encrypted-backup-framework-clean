# Flet Server GUI Documentation

This folder contains documentation for the Flet Server GUI components and systems.

## Available Documentation

- [`unified_theme_system.md`](unified_theme_system.md) - Complete documentation for the consolidated Material Design 3 theme system

## Purpose

This documentation serves to:
1. Explain the architecture and usage of key systems
2. Provide migration guides for developers
3. Document best practices and patterns
4. Serve as a reference for future development

## Note

Validation and test files in the root directory (`validate_*.py`) are kept for ongoing testing and verification purposes. These are not archived as they provide valuable test coverage for the system.
