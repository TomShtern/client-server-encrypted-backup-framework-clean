# Buttons

Flet offers a comprehensive set of buttons to be used in your applications.

Here is a list of the button controls available in Flet:

*   **Material Design Buttons**: These are based on Google's Material Design system.
    *   `ElevatedButton`: A button with a shadow that appears to be "elevated".
    *   `FilledButton`: A contained button with a background color.
    *   `FilledTonalButton`: A contained button with a lighter background color.
    *   `OutlinedButton`: A button with a border and no background color.
    *   `TextButton`: A button with no border or background, just text.
*   **Specialty Buttons**:
    *   `FloatingActionButton`: A circular button that "floats" above the UI, typically used for a primary action.
    *   `IconButton`: A button that displays an icon instead of text.
    *   `PopupMenuButton`: A button that displays a menu when clicked.
    *   `MenuItemButton`: Represents an item within a `PopupMenuButton`.
    *   `SegmentedButton`: A set of buttons that allow the user to select one option from a set.
    *   `SubmenuButton`: A button that opens a submenu.
