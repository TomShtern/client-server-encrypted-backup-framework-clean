# Utility Controls

This page provides an overview of the utility controls available in the Flet framework. These controls offer a wide range of functionalities, from interacting with device hardware to managing user gestures and handling accessibility features.

Here is a summary of the available controls:

*   **Media:** `Audio` for playing sounds, `AudioRecorder` for capturing audio, and `Video` for displaying videos.
*   **Device Features:** `Flashlight` to control the camera flash, `Geolocator` to get the device's location, `HapticFeedback` for physical feedback, and `ShakeDetector` to react to device shakes.
*   **User Interaction:** `Draggable` and `DragTarget` for drag-and-drop operations, `GestureDetector` for capturing complex user gestures, `InteractiveViewer` for pan and zoom functionality, and `SelectionArea` to enable text selection.
*   **Files:** `FilePicker` allows users to select files from their device.
*   **Windowing and App Integration:** `FletApp` to embed one Flet app within another, and `WindowDragArea` to create custom draggable window regions.
*   **Permissions and Accessibility:** `PermissionHandler` to manage app permissions, and a suite of `Semantics` controls to improve accessibility.
*   **Advanced UI:** `ShaderMask` for applying visual effects and `TransparentPointer` for controlling gesture behavior in stacked widgets.
