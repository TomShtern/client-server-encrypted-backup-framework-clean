# Encrypted Backup System - User Guide

This document explains how the encrypted backup system works in simple, non-technical terms.

## What Does This System Do?

The Encrypted Backup System is like a **secure digital vault** that protects your important files by:
1. **Encrypting** your files (scrambling them so only you can read them)
2. **Transferring** them safely to a backup server
3. **Storing** them securely for safekeeping

Think of it as a high-security armored truck that picks up your valuable documents, locks them in an unbreakable safe, and delivers them to a secure storage facility.
