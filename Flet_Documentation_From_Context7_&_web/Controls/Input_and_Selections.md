# Input and Selections

This document provides an overview of the input and selection controls available in the Flet framework. These controls are used to build interactive forms and capture user input in a Flet application.

Here is a list of the available controls:

*   **AutoComplete**: Helps users make a selection by providing a list of suggestions as they type.
*   **AutofillGroup**: Groups autofillable controls like `TextField`.
*   **Checkbox**: Allows for the selection of one or more items from a group.
*   **Chip**: A compact element that can represent an attribute, text, entity, or action.
*   **Dropdown**: A dropdown menu for selecting a single item from a list.
*   **DropdownM2**: A Material Design button for selecting from a list of items.
*   **Radio**: Allows for the selection of a single option from a group of two or more.
*   **RangeSlider**: A slider for selecting a range of values.
*   **SearchBar**: A Material Design search bar.
*   **Slider**: A slider for selecting a single value from a range.
*   **Switch**: A toggle switch for choosing between two mutually exclusive options.
*   **TextField**: A standard text input field.
