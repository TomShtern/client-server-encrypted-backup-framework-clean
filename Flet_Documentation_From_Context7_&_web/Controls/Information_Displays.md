# Information Displays

This page lists various UI controls available in the Flet framework for displaying information. These controls include:

*   **Ads**: For displaying advertisements.
*   **Canvas**: For custom drawing of shapes and graphics.
*   **CircleAvatar**: To represent a user with a circular avatar.
*   **Icon**: For displaying Material icons.
*   **Image**: For displaying images.
*   **Map**: For displaying maps.
*   **Markdown**: To render text in Markdown format.
*   **Text**: For displaying plain text.
*   **ProgressBar**: A linear progress indicator.
*   **ProgressRing**: A circular progress indicator.
*   **WebView**: To load and display web pages within the app.
